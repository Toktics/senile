# Global Export Standards

- Colour profile: sRGB
- No embedded colour profiles
- Avoid heavy grain
- Keep assets under 800KB where possible
- UI text layered in code, not baked into images

---

# Production Philosophy

Mechanics are code-driven.
Assets enhance atmosphere.
Visual tone: institutional, serious, subtly absurd.
