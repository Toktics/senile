## Export Guidelines

- Preferred: `AVIF` or `WEBP`
- Width target: 900px max
- Keep file size ideally under 350KB per card
- Preserve vertical card aspect ratio
- Use transparent background where available for better blend in archive UI

If you use different filenames, update `src/content/archive-data.ts`.
